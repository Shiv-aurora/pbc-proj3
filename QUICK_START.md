# Wellness App - Quick Start

## First Time Setup

### 1. Clone the Repository

```bash
git clone https://github.com/Shiv-aurora/pbc-proj3.git
cd pbc-proj3
```

### 2. Install Dependencies

```bash
npm install
```

This will install all required packages listed in `package.json`:
- Angular 18
- Chart.js
- TypeScript
- And all other dependencies

### 3. Run the App

```bash
npm start
```

Open: **http://localhost:4200/**

## What It Is

Habit tracker with warm design (beige/cream colors, Playfair Display font).

**Features:**
- Add daily habits
- Check them off
- See 7-day progress charts
- Auto-saves everything

## Test It

1. Click "+ Add New Habit"
2. Add "Drink Water"
3. Check it off ☑️
4. See the chart update
5. Refresh page (data persists!)
